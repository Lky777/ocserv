___
Usage  
```
apt-get update
apt-get upgrade
apt-get install wget
wget https://git.io/JIume --no-check-certificate -O ocservauto.sh
bash ocservauto.sh
```

Restart  
```
/etc/init.d/ocserv restart  #when the Profiles(/etc/ocserv/) is changed 
```

Help  
```
bash ocservauto.sh h
```
